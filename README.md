# glottopos: integrating geographic and linguistic methods for the use of toponyms (place names) in language history

Toponyms (place names) can be a powerful type of data for learning about language histories.
The functions in the glottopos repository are associated with the following paper:

Emlen, N.Q. and Norder, S.J. (2026) Integrating linguistic and geographic methods in toponymic analysis The case of Puquina (Central Andes). Diachronica
